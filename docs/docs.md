# Dptpy Docs
Deplux 비공식 설정 도구

# 명령어들(옵션)
## --init/ --i / --setup

배포를 위한 Dockerfile을 생성합니다


## --new / --n / --create

새로운 deplux 프로젝트를 만듭니다

## --zip / --z/  --build

명령어를 실행한 루트 디렉터리를 Deplux용 호스팅 파일로 압축합니다

## --krhelp / --kh

이 도움말 메세지를 출력합니다(KR)
